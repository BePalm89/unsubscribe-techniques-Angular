@UntilDestroy decorator solution

Implementation

1. Install @ngneat/until-destroy package

2. Using a simple @decorator in the component

3. Using the operator UntilDestroyed Rxjs operators

Advantages/Disadvantages

+ Cleaner solution

+ Operator marks the subscriptions to be unsubscribed from when the component gets destroyed by Angular runtime

+ Do not need to implements the ngOnDestroy life cycle hook

- Need to install a third-party package

- Works for Angular9+
