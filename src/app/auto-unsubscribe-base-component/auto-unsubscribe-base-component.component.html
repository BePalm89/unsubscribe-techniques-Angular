Auto-Unsubscribe Base component

Implementation

1. Create a class called AutoUnsubscribeComponent in the share module

2. In the component where we want to unsubscribe extends the AutoUnsubscribeComponent

3. Called the method addSubscriptions()

4. We must se the super.ngOnDestroy

Advantages/Disadvantages

+ It is DIY pure Angular + RxJS approach

- Using inheritance ( TS is a single-inheritance language )

- Need of implementing ngOnDestroy
