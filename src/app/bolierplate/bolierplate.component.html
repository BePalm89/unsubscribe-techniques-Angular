Boilerplate solution

Implementation

1. Declare of all the subscription we need

2. Use the subscribe() method in order to notify the observable

3. Use the unsubscribe() method in order to complete the observable

Advantages/Disadvantages

- If we do not unsubscribe the observable execution is going to be infinite

- We need to remember to implements the ngOnDestroy lifecycle hook

- Memory leak in JS runtime
